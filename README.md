# bias-variance-visualization
Visualization of bias-variance tradeoff and overfitting using polynomial regression

## Problem
Understanding underfitting and overfitting in machine learning models.

## Approach
Used polynomial regression with different degrees to visualize model behavior.

## Results
- Degree 1: Underfitting
- Degree 2: Good fit
- Degree 10: Overfitting

## Key Learnings
- High bias leads to underfitting
- High variance leads to overfitting
